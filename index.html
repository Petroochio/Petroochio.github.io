A Scanner Orderly

Tags: ArduinoC++JavaScriptNode.js

A 3D printed coaster that enables patrons to order drinks remotely. This was a project for the Physical Computing class I took last spring. Using an RGB sensor to detect the color underneath it, users would pick an item from a color coded menu, the order was then sent to a node.js server that updated a web-interface designed for the bartender to keep track of orders.

Mouse Rat

Tags: ArduinoC++

A mouse embedded into a glove that is controlled with accelerometers. This was also a project for the Physical Computing class I took last spring. One of my partners for this project had a history of carpal tunnel disease so we decided to try and make a more comfortable mouse. Using a sewable Arduino floura, an accelerometer, and capacitve touch sensors we made a wearable mouse.

Stargazer

Tags: JavaScriptHTML5 Canvas

A web-app utilizing a HTML5 Canvas library, Starchartjs, to render a view of the night sky based on cached data of star charts. It hooks into the lookUP API to find the locations of heavenly bodies the user searches for. Starchartjs allowed for rendering of constellations. On start the application shows you the night sky at the user's GPS location, then the user can scroll through all latitue/longitudes.
